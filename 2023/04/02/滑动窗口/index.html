滑动窗口

滑动窗口（sliding window）是一种算法技术，通常用于数组或字符串的问题中。滑动窗口算法通过维护一个窗口，该窗口在数组或字符串上移动，并在每次移动时更新窗口的内容，以便有效地解决一些问题。

具体来说，滑动窗口算法通常涉及三个指针：左指针，右指针和窗口指针。左指针指向当前窗口的左侧边界，右指针指向当前窗口的右侧边界，而窗口指针则用于移动窗口并更新答案。在每次移动窗口时，我们可以根据当前窗口的状态来更新答案，并在不改变窗口大小的情况下将窗口向右移动一位。

滑动窗口算法通常用于解决需要在一个连续子数组或子串中查找特定属性或计算某些统计数据的问题，例如求最小或最大值、求和、计数或检测满足特定条件的子数组或子串等。

本文采用C#代码做解释

题目：

给定一个字符串 s ，请你找出其中不含有重复字符的 最长子串 的长度。

public class Solution {
    public int LengthOfLongestSubstring(string s) {
    int n = s.Length;
    HashSet set = new HashSet();
    //这里使用哈希表而不是简单数组就是因为方便查找，时间复杂度为O(1)
    int ans = 0, i = 0, j = 0;
    while (i < n && j < n) {
        if (!set.Contains(s[j])) {
            set.Add(s[j++]);
            ans = Math.Max(ans, j - i);
        } else {
            set.Remove(s[i++]);
        }
    }
    return ans;
}

}

/*HashSet 是 C# 中的一个泛型集合类，用于存储不重复的元素。在这个例子中，HashSet 就是一个存储不重复字符的集合。在代码中，我们使用 HashSet 来判断字符是否出现过，以及在滑动窗口中添加或删除字符。

HashSet 类提供了许多有用的方法，如 Add、Remove、Contains 等等，用于添加、删除、判断元素是否存在等操作。当需要存储一些元素，且不希望出现重复元素时，可以使用 HashSet 来实现。*/

注：

这里用了一个 HashSet 来存储当前滑动窗口中的字符。i 和 j 分别表示滑动窗口的左右边界，ans 表示当前最长不含重复字符子串的长度。在循环中，如果右边界所在的字符不在 HashSet 中，则将其添加到 HashSet 中，并将右边界右移一位。此时，由于新添加的字符不重复，因此更新 ans 的值。如果右边界所在的字符已经在 HashSet 中，说明当前窗口中有重复的字符，需要将左边界右移一位，并将左边界所在的字符从 HashSet 中移除，直到窗口中不再有重复的字符为止。

这种方法的时间复杂度为 O(n)，空间复杂度为 O(min(m, n))，其中 m 表示字符集的大小。
