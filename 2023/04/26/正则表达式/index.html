正则表达式

特殊字符 描述
. 匹配除了“\n”之外的任意单个字符。要匹配包括“\n”在内的任意字符，使用”[.\n]”
\d 匹配一个数字字符，等价于[0-9]
\D 匹配一个非数字字符，等价于[^0-9]
\s 匹配任意空白字符，包括空格、制表符、换页符等，等价于[\f\n\r\t\v]
\S 匹配任意非空白字符，等价于[^\f\n\r\t\v]
\w 匹配包括下划线的任意单词字符，等价于[A-Za-z0-9_]
\W 匹配任意非单词字符，等价于[^A-Za-z0-9_]

在正则表达式中，要匹配可变长度的字符：

  • 用*表示任意个数的字符（包括0）
  • 用+表示至少一个字符
  • 用？表示0或者一个字符
  • 用{ }表示可以出现的次数

例如：

\d{3}\s+\d{3,8}就匹配电话号码

如果要匹配像010-123456这样的，要使用转义字符

以下是需要进行转义的特殊字符的比较完整的列表：

  1. . （点号）：表示匹配任意单个字符，需要使用 \. 来匹配字面上的点号。
  2. * （星号）：表示匹配前一个字符零次或多次，需要使用 \* 来匹配字面上的星号。
  3. + （加号）：表示匹配前一个字符一次或多次，需要使用 \+ 来匹配字面上的加号。
  4. ? （问号）：表示匹配前一个字符零次或一次，需要使用 \? 来匹配字面上的问号。
  5. { 和 } （左花括号和右花括号）：表示限定匹配次数的范围，例如 {n} 表示匹配前一个字符恰好 n 次，需要使用 \{ 和 \} 来匹配字面上的花括号。
  6. | （竖线）：表示逻辑或（或者），需要使用 \| 来匹配字面上的竖线。
  7. [ 和 ] （左方括号和右方括号）：表示字符范围，例如 [a-z] 表示匹配小写字母 a 到 z 中的任意一个字符，需要使用 \[ 和 \] 来匹配字面上的方括号。
  8. ( 和 ) （左括号和右括号）：用于分组和捕获匹配，需要使用 \( 和 \) 来匹配字面上的括号。
  9. { 和 } （左大括号和右大括号）：用于表示量词或者限定符的特定重复次数，例如 {n} 表示匹配前一个字符恰好 n 次，需要使用 \{ 和 \} 来匹配字面上的大括号。
  10. \ （反斜杠）：用于转义特殊字符，例如 \., \*, \[ 等，需要使用 \\ 来匹配字面上的反斜杠。
  11. - （减号）：用于表示字符范围，需要使用 \- 来匹配字面上的减号。
  12. 但在某些正则表达式引擎中，下划线可能被视为特殊字符，因此为了避免可能的歧义，可以将其转义为 \_

其它：

  • |表示或运算，例如：（ab）|（cd）
  • ^匹配行首
  • $匹配行尾
  • <.+>默认贪婪匹配
  • <.+?>懒惰匹配

Python中的正则表达式

re.match

re.match()函数返回一个match对象，如果正则表达式在字符串的开头匹配成功；如果匹配失败，则返回None。

match对象是一个包含匹配结果的对象，它具有以下方法可以用于获取匹配到的信息：

  • group(): 返回匹配到的内容，可以传入一个可选的参数表示要获取的子组序号，默认为0，表示整个匹配到的字符串。
  • start(): 返回匹配到的子串在原字符串中的起始位置的索引。
  • end(): 返回匹配到的子串在原字符串中的结束位置的索引。
  • span(): 返回一个包含匹配到的子串在原字符串中起始位置和结束位置索引的元组。

以下是一个示例，展示如何使用match对象获取匹配到的信息：

import re

# 定义一个正则表达式模式
pattern = r'hello'
# 要进行匹配的字符串
string = 'hello world'

# 进行正则表达式匹配
match_obj = re.match(pattern, string)

# 判断匹配结果
if match_obj:
    print('匹配成功！')
    print('匹配到的内容：', match_obj.group(0))
    print('匹配到的内容起始位置：', match_obj.start())
    print('匹配到的内容结束位置：', match_obj.end())
    print('匹配到的内容起始和结束位置：', match_obj.span())
else:
    print('匹配失败！')

re.search

re.search()函数的使用方式如下：

re.search(pattern, string, flags=0)

其中，pattern是正则表达式模式，用于匹配字符串；string是待匹配的字符串；flags是可选参数，用于指定正则表达式的匹配标志。

以下是re.search()函数的参数说明：

  • pattern：正则表达式模式，用于指定需要匹配的字符串的规则。可以是字符串形式的正则表达式，也可以是由re.compile()函数编译得到的正则表达式对象。

  • string：待匹配的字符串。

  • flags ：可选参数，用于指定正则表达式的匹配标志。常用的标志包括：

    • re.IGNORECASE：忽略大小写匹配。
    • re.MULTILINE：多行模式匹配。
    • re.DOTALL：允许 . 匹配换行符。
    • re.UNICODE：启用 Unicode 匹配。
    • re.VERBOSE：忽略正则表达式中的空白字符和注释。

re.search()函数的返回值是一个re.Match对象，表示第一个匹配的结果。re.Match对象具有一系列方法和属性，可以用于获取匹配结果的具体信息，如匹配的字符串、匹配的位置、匹配的分组等。

以下是一个简单的示例，演示了如何使用re.search()函数进行正则表达式匹配：

import re

# 定义正则表达式模式
pattern = r'hello'

# 待匹配的字符串
string = 'hello world!'

# 使用re.search()函数进行匹配
match = re.search(pattern, string)

# 判断是否匹配成功
if match:
    print('匹配成功！')
    # 获取匹配的结果
    print('匹配的字符串：', match.group())
    print('匹配的位置：', match.start(), '-', match.end())
else:
    print('匹配失败！')

(题外话：“：=”

在 Python 中，”:=” 是一个可变表达式的赋值语法，也被称为 “walrus operator”（海象运算符）。它的引入是为了在表达式中进行赋值操作，以简化代码并增加可读性。

使用 “:=” 运算符，可以在一行中同时进行变量赋值和条件判断，例如在 while 循环中使用：

# 示例：使用 ":=" 在 while 循环中进行赋值和条件判断
while (line := input()) != "quit":
    print("输入的内容是:", line)

)
