Markdown的使用

1.标题

就是6个标题种类，#越多标题越小

2.引用

用一个>开头就行，比如：

这是一段引用

3.无序列表

前面加上-和空格就行，例如：

  • 这是一个无序列表

或者用*加空格，比如：

  • 这是一个无序列表

4.任务列表

就是- [ ] ，注意这里有三个空格

  • 这是一个任务列表

5.代码块

三个反引号~~~加上语言种类，例如：

using System;
class WH{
    public static void Main()
    {
        Console.WriteLine("WHNB!")
    }
}

6.数学公式

四个$符号，$$ $$其中加上想要加上的内容，(直接打上两个$然后回车好像也行），用LaTex，例如：
$$
\frac{\partial f}{\partial x}=2\sqrt{a}x
$$

7.表格

用|隔开，像这样：“|姓名|年龄|成绩|”

还可以左对齐右对齐：就是在下一行写“|：—|：—：|—：|“

短横线至少有一个，当然也可以直接生成表格后调整

姓名 年龄 成绩

8.脚注

就是加上**[^文本内容]**

这是一个脚注^脚注真

[百度](baidu.com “这是百度”)

11.引用链接

\[百度][id]

[id]:google.com “一个搜索引擎”

这样就可以多次引用
