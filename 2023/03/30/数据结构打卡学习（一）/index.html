数据结构打卡学习（一）

本文采用C#语言

第一题题目描述：

给你一个整数数组 nums 。如果任一值在数组中出现 至少两次 ，返回 true ；如果数组中每个元素互不相同，返回 false 。

解答1:(使用Linq去重)

public class Solution {
    public bool ContainsDuplicate(int[] nums) {
        int[] nums1=nums.Distinct().ToArray();
        //这是Linq库中的Distinct()方法去除数组中的重复元素
        if(nums1.Length==nums.Length) return false;
        //如果长度一致就说明没有重复元素
        else return true;
    }
}

解答2：（使用HashSet去重）

public class Solution {
    public bool ContainsDuplicate(int[] nums) {
        HashSet hashset=new HashSet(nums);
        int[] result=hashset.ToArray();
        if(result.Length==nums.Length) return false;
        else return true;
    }
}

下面是官方进阶的分治法(时间复杂度O(nlogn))：

public int MaxSubArray(int[] nums) {
    return DivideAndConquer(nums, 0, nums.Length - 1);
}

private int DivideAndConquer(int[] nums, int left, int right) {
    if (left == right) {
        return nums[left];
    }
    //数组中只有一个元素

    int mid = left + (right - left) / 2;
    int leftMaxSum = DivideAndConquer(nums, left, mid);
    int rightMaxSum = DivideAndConquer(nums, mid + 1, right);
    int crossMaxSum = CrossMaxSubArray(nums, left, right, mid);

    return Math.Max(Math.Max(leftMaxSum, rightMaxSum), crossMaxSum);
}

private int CrossMaxSubArray(int[] nums, int left, int right, int mid) {
    int leftSum = int.MinValue;
    //为什么会是int.MinValue，后面会有注释说明
    int sum = 0;
    for (int i = mid; i >= left; i--) {
        sum += nums[i];
        leftSum = Math.Max(leftSum, sum);
    }

    int rightSum = int.MinValue;
    sum = 0;
    for (int i = mid + 1; i <= right; i++) {
        sum += nums[i];
        rightSum = Math.Max(rightSum, sum);
    }

    return leftSum + rightSum;
}

/*执行用时：
180 ms, 在所有 C# 提交中击败了54.99%的用户
内存消耗：
49.2 MB, 在所有 C# 提交中击败了10.39%的用户
都是些什么神仙。。。*/

注：

分治法常用于求解一些可以分成若干个相同的子问题的问题，例如排序、查找、最大子数组和等。

关于int.MinValue的注解：

在求解跨越中点的最大子数组问题时，初始的最大和需要设置为一个非常小的数，以保证后续的比较操作不会失效。

在上面的代码中，将 leftSum 的初始值设置为 int.MinValue，是为了保证 leftSum 在接下来的比较中能够被正确更新。如果将 leftSum 的初始值设置为 0 或其他正数，则可能会导致 leftSum 在接下来的比较中无法被正确更新。
